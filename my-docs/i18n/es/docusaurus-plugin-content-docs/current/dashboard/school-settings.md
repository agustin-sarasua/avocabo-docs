# Configuración de la Escuela

La configuración de la escuela permite a los administradores personalizar PragmaTeach para su institución. Esto incluye marca, gestión de suscripciones y configuración organizacional.

:::note Solo Administrador
La configuración de la escuela solo es accesible para usuarios con el rol de Administrador.
:::

## Acceder a la Configuración de la Escuela

1. Navega a **Configuración de la Escuela** en la barra lateral
2. O haz clic en el icono de engranaje ⚙️ en la navegación superior

---

## Marca

### Logo de la Escuela

Sube el logo de tu escuela:
1. Ve a la sección **Marca**
2. Haz clic en **Subir Logo**
3. Selecciona un archivo de imagen (PNG, JPG, SVG)
4. El logo aparece en el panel y la aplicación para estudiantes

**Recomendaciones:**
- La relación de aspecto cuadrada funciona mejor
- Mínimo 200x200 píxeles
- Fondo transparente (PNG) preferido

### Nombre de la Escuela

Establece el nombre para mostrar de tu escuela:
1. Ingresa el nombre de la escuela
2. Esto aparece en encabezados y reportes
3. Haz clic en **Guardar**

### Colores Personalizados

Algunos planes permiten temas de colores personalizados:
- Color primario
- Color secundario
- Color de acento

## Información de Suscripción

### Plan Actual

Ver los detalles de tu suscripción:

| Información | Descripción |
|-------------|-------------|
| **Tipo de Plan** | Gratis, Básico, Pro, Empresarial |
| **Estado** | Activo, Prueba, Expirado |
| **Fecha de Renovación** | Próxima fecha de facturación |
| **Créditos Restantes** | Créditos de generación con IA |

### Uso de Créditos

Monitorea tu consumo de créditos:

| Acción | Créditos Usados |
|--------|--------------|
| **Generación de Curso** | Varía según tamaño |
| **Creación de Ejercicios** | Por ejercicio |
| **Traducción** | Por pieza de contenido |
| **Consultas del Asistente de IA** | Por conversación |

### Actualizar Planes

Para actualizar:
1. Haz clic en **Actualizar Plan**
2. Revisa los planes disponibles
3. Selecciona nuevo plan
4. Completa el pago

## Estadísticas de Uso

### Uso de Toda la Escuela

Ver estadísticas agregadas:
- Total de cursos creados
- Total de lecciones generadas
- Usuarios activos este mes
- Almacenamiento usado

### Actividad del Profesor

Rastrea la participación del profesor:
- Cursos creados por profesor
- Lecciones activadas
- Interacciones con estudiantes

### Participación de Estudiantes

Monitorea el uso de estudiantes:
- Estudiantes activos
- Lecciones completadas
- Tiempo promedio dedicado

## Límites de Usuarios

### Límites Actuales

Dependiendo de tu plan:

| Plan | Estudiantes | Profesores | Cursos |
|------|----------|---------|---------|
| **Gratis** | 25 | 2 | 3 |
| **Básico** | 100 | 5 | 10 |
| **Pro** | 500 | 25 | Ilimitados |
| **Empresarial** | Ilimitados | Ilimitados | Ilimitados |

### Acercándose a Límites

Recibirás notificaciones cuando:
- Se alcance el 80% del límite de usuarios
- 90% de uso de créditos
- Se acerque el límite de almacenamiento

## Configuración de Seguridad

### Autenticación

Configura opciones de inicio de sesión:
- Correo/Contraseña (predeterminado)
- Inicio de Sesión con Google

### Políticas de Contraseña

- Expiración de contraseña

## Notificaciones

### Notificaciones por Correo

| Notificación | Destinatarios | Predeterminado |
|--------------|------------|---------|
| **Bienvenida de Usuario Nuevo** | Usuarios nuevos | ✅ Activado |
| **Progreso Semanal** | Profesores | ⚙️ Opcional |
| **Actualizaciones del Sistema** | Administradores | ✅ Activado |
| **Alertas de Uso** | Administradores | ✅ Activado |

### Notificaciones en la Aplicación

Controla notificaciones del panel:
- Nueva actividad de estudiantes
- Finalizaciones de cursos
- Anuncios del sistema

## Gestión de Datos

### Exportación de Datos

Exporta los datos de tu escuela:
1. Ve a **Gestión de Datos**
2. Selecciona tipo de datos para exportar
3. Elige formato (CSV, JSON)
4. Descarga el archivo

**Exportaciones disponibles:**
- Lista de usuarios
- Estructuras de cursos
- Datos de progreso
- Registros de actividad

### Retención de Datos

Configura cuánto tiempo se mantienen los datos:
- Historial de progreso de estudiantes
- Registros de actividad
- Contenido eliminado

## Solución de Problemas

### El Logo No Se Muestra

- Verifica formato de archivo (PNG, JPG, SVG)
- Verifica tamaño de archivo (menos de 5MB)
- Prueba un navegador diferente
- Limpia la caché

### Créditos Agotándose

- Revisa patrones de uso
- Optimiza la generación de cursos
- Considera actualizar el plan
- Contacta ventas para opciones

### Problemas de Acceso

Si los usuarios no pueden acceder:
- Verifica que los límites de usuarios no se hayan excedido
- Verifica el estado de la suscripción
- Revisa permisos de usuario
