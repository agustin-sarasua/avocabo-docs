# Crear Cursos con IA

PragmaTeach usa inteligencia artificial para ayudarte a crear cursos completos a partir de tus materiales educativos. Esta guía te lleva a través de todo el proceso de creación de cursos.

## Descripción General

El proceso de creación de cursos sigue estos pasos:

```
1. Información Básica → 2. Duración e Idiomas → 3. Documentos y Programa → 4. Revisar y Generar
```

:::tip Entrada de Calidad = Salida de Calidad
¡Cuanto mejor sea tu configuración inicial (documentos, programa del curso, descripciones), mejor será el contenido generado por la IA!
:::

## Paso 1: Información Básica

### Tipo de Curso

Primero, selecciona el tipo de curso que estás creando:

| Tipo | Mejor Para |
|------|----------|
| **Curso de Idioma** | Enseñar idiomas extranjeros (Inglés, Español, Alemán) |
| **Curso Académico** | Cursos específicos de materias (Matemáticas, Ciencias, Historia, etc.) |

### Asignatura

Selecciona o ingresa una asignatura:
- Para **Cursos de Idioma**: Inglés, Español, Alemán, etc.
- Para **Cursos Académicos**: Matemáticas, Ciencias, Historia, o ingresa una asignatura personalizada

### Título del Curso

Elige un título claro y descriptivo:

✅ **Buenos ejemplos:**
- "Matemáticas de 7º Grado: Álgebra y Geometría"
- "Español Intermedio: Conversación y Gramática"
- "Introducción a la Biología: Células y Organismos"

❌ **Evita:**
- "Matemáticas" (demasiado vago)
- "Mi Curso" (no descriptivo)

### Descripción del Curso

Incluye:
- Objetivos de aprendizaje
- Temas clave cubiertos
- Resultados objetivo

**Ejemplo:**
> "Este curso cubre conceptos fundamentales de álgebra para estudiantes de 7º grado, incluyendo ecuaciones lineales, geometría básica y estrategias de resolución de problemas. Los estudiantes desarrollarán habilidades de razonamiento matemático y aprenderán a aplicar conceptos a problemas del mundo real."

### Audiencia Objetivo

Especifica el nivel educativo:
- Escuela Primaria (edades 6-11)
- Escuela Secundaria (edades 11-14)
- Escuela Preparatoria (edades 14-18)
- Universidad/Adultos

:::info Por Qué Esto Importa
La IA usa esta información para generar contenido apropiado para la edad, vocabulario y dificultad de ejercicios.
:::

### Asignar Grupos

Selecciona qué grupos de estudiantes tendrán acceso a este curso:
- Busca y selecciona de grupos existentes
- O haz el curso público para todos los estudiantes

---

## Paso 2: Duración y Estructura

### Duración del Curso

Elige el número de lecciones:

| Opción | Lecciones | Recomendado Para |
|--------|---------|-----------------|
| Mini | ~10 lecciones | Temas cortos, cursos de repaso |
| Corto | ~20 lecciones | Cursos de un trimestre |
| 3 Meses | ~30 lecciones | Cursos de semestre |
| 6 Meses | ~60 lecciones | Cursos de año completo |
| Personalizado | Tú defines | Requisitos específicos |

### Configuración de Idiomas

#### Idioma Principal
El idioma principal para el contenido del curso. Esto es lo que los estudiantes verán principalmente.

#### Idiomas Secundarios
Habilita la traducción automática a idiomas adicionales:
- Español
- Inglés
- Alemán

#### Porcentaje de Traducción
Para cursos bilingües o de idiomas, establece qué porcentaje de lecciones aparecen en el idioma secundario:
- 0% = Todo el contenido en idioma principal
- 50% = La mitad de las lecciones en cada idioma
- 100% = Todo el contenido en idioma secundario

---

## Paso 3: Documentos y Programa del Curso

### Subir Documentos de Referencia

Los documentos proporcionan contexto para la generación de contenido con IA. La IA busca en estos documentos al crear lecciones.

**Formatos soportados:**
- Archivos PDF
- Archivos DOCX
- URLs web

**Mejores prácticas:**
1. ✅ Sube solo documentos relevantes y de alta calidad
2. ✅ Asegúrate de que los documentos estén completos y bien formateados
3. ✅ Mejor tener 2-3 documentos excelentes que 10 mediocres
4. ✅ Espera a que los documentos muestren estado "indexado" antes de continuar

**Qué subir:**
- Libros de texto
- Guías de estudio
- Documentos de currículo oficiales
- Artículos relevantes

### Programa del Curso (Currículo)

Para cumplir con estándares educativos oficiales, sube tu documento de currículo:

- Documentos de programa oficiales del ministerio/institución
- Los documentos con tablas de contenido funcionan mejor
- Incluye objetivos de aprendizaje y competencias

**La IA extrae:**
- Resumen del programa
- Objetivos de aprendizaje
- Estructura de contenido
- Códigos de competencia (CE)
- Criterios de logro

:::warning Importante
Verifica que el contenido extraído sea preciso antes de continuar. Esto asegura que tu curso se alinee con los estándares requeridos.
:::

---

## Paso 4: Revisar el Plan de Agrupación

Antes de generar contenido, la IA analiza tus documentos y propone una estructura de contenido.

### Lo Que Muestra el Plan de Agrupación

- **Módulos** - Secciones principales del curso
- **Lecciones por Módulo** - Cómo se distribuye el contenido
- **Orden del Contenido** - Progresión de temas fundamentales a avanzados

### Lista de Verificación de Revisión

- [ ] El número de módulos tiene sentido para el tema
- [ ] Las lecciones están distribuidas uniformemente entre módulos
- [ ] Los conceptos fundamentales vienen antes de los temas avanzados
- [ ] No hay duplicación innecesaria de contenido
- [ ] Flujo lógico para el aprendizaje de los estudiantes

### Ajustar el Plan

Si es necesario, puedes:
- Reorganizar contenido entre módulos
- Fusionar o dividir módulos
- Reordenar lecciones dentro de módulos

---

## Paso 5: Generar el Curso

Una vez que hayas revisado todo:

1. Haz clic en **"Generar Curso"**
2. Espera a que la IA cree el contenido (esto puede tomar varios minutos)
3. Ve actualizaciones de progreso mientras se generan módulos y lecciones

### Lo Que Se Genera

Para cada **Módulo:**
- Título y descripción
- Temas cubiertos
- Imagen del módulo

Para cada **Lección:**
- Título y descripción
- Temas de la lección
- Imagen de la lección
- Conexión con el contenido del documento

Para cada **Ejercicio:** (generado por separado)
- Preguntas de opción múltiple
- Comprensión lectora
- Comprensión auditiva
- Ejercicios de video
- Problemas matemáticos
- Actividades interactivas

## Después de Crear el Curso

### Revisar Contenido Generado

Antes de activar:
1. ✅ Revisa títulos y descripciones de módulos
2. ✅ Verifica el contenido de las lecciones por precisión
3. ✅ Prueba el flujo del curso
4. ✅ Verifica traducciones si aplica

### Asignar Profesores

Agrega profesores adicionales para ayudar a gestionar el curso:
1. Ve a Detalles del Curso
2. Haz clic en "Gestionar Profesores"
3. Selecciona profesores para agregar

### Activar el Curso

El curso se crea en estado **inactivo**. Para hacerlo visible a los estudiantes:

1. Revisa todo el contenido
2. Haz clic en **"Activar Curso"**
3. Los estudiantes en grupos asignados ahora pueden verlo

### Activar Lecciones

Controla qué lecciones pueden acceder los estudiantes:

- **Activar individualmente**: Activa lecciones una por una
- **Activar en lote**: Usa el widget del panel para activar múltiples lecciones
- **Recomendado**: Activa progresivamente mientras los estudiantes completan lecciones anteriores

## Consejos para Mejores Resultados

### Calidad de Documentos
| Hacer | No Hacer |
|----|-------|
| Subir documentos completos y oficiales | Subir documentos incompletos o borradores |
| Usar PDFs bien formateados | Usar imágenes escaneadas con OCR deficiente |
| Incluir contenido completo | Incluir solo resúmenes |

### Información del Curso
| Hacer | No Hacer |
|----|-------|
| Proporcionar descripciones detalladas | Usar títulos de una palabra |
| Especificar audiencia exacta | Ser vago sobre el nivel del estudiante |
| Incluir objetivos de aprendizaje | Omitir la descripción |

### Proceso de Revisión
| Hacer | No Hacer |
|----|-------|
| Revisar el plan de agrupación cuidadosamente | Aceptar sin verificar |
| Verificar contenido antes de activar | Activar inmediatamente después de la generación |
| Probar la experiencia del estudiante | Omitir las pruebas |

---

## Solución de Problemas

### La Generación del Curso Toma Demasiado Tiempo
- Los cursos grandes (60+ lecciones) toman 15-30 minutos
- Si se queda atascado, intenta actualizar la página
- Verifica tu conexión a internet

### El Contenido Generado Parece Incorrecto
- Revisa tus documentos subidos por relevancia
- Verifica si el programa del curso se extrajo correctamente
- Considera proporcionar descripciones más detalladas

### Los Documentos No Se Están Indexando
- Asegúrate de que el formato de archivo sea compatible
- Intenta volver a subir el documento
- Contacta al soporte si el problema persiste
