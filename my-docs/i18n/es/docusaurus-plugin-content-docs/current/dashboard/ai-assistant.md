# Asistente de IA para Profesores

El Asistente de IA para Profesores es tu ayudante inteligente que puede responder preguntas sobre tus estudiantes, cursos, lecciones y proporcionar recomendaciones de enseñanza basadas en datos reales.

## ¿Qué Es?

El Asistente de IA para Profesores usa los datos de tu escuela para proporcionar:
- **Información sobre estudiantes** - Análisis de rendimiento y recomendaciones
- **Análisis de cursos** - Efectividad y sugerencias de mejora
- **Reportes rápidos** - Generar reportes bajo demanda
- **Apoyo docente** - Responder preguntas y proporcionar orientación

---

## Acceder al Asistente

### Desde el Panel

1. Haz clic en el botón flotante del icono **✨**
2. O navega a **Chat** en la barra lateral
3. Se abre la ventana de chat

### Durante Cualquier Pantalla

El botón flotante está disponible en todo el panel para acceso rápido.

## Tipos de Preguntas

### Sobre Estudiantes

Pregunta sobre estudiantes individuales o grupos:

**Preguntas de rendimiento:**
```
"¿Cómo está rindiendo María García en Matemáticas 7?"

```

```
"¿Qué estudiantes necesitan ayuda extra en Ciencias?"
```

"¿Qué lecciones tienen las tasas de finalización más bajas?"

```
"¿Qué temas son los más difíciles para los estudiantes?"

```

```
"¿Cuál es el tiempo promedio que los estudiantes dedican a la Lección 7?"

## Consejos para Preguntas Efectivas

### Sé Específico

| Bueno | Por Qué Mejor | Malo |
|------|------------|-----|
| "¿Cómo está Juan Pérez en Matemáticas 7?" | Estudiante y curso específicos | "¿Cómo están los estudiantes?" |
| "¿Qué estudiantes en 7A no han completado la Lección 5?" | Grupo y lección específicos | "¿Quién está atrasado?" |
| "Compara el progreso entre 7A y 7B en Ciencias" | Comparación clara | "¿Qué clase es mejor?" |

### Proporciona Contexto

Al preguntar, incluye:
- **Nombre del curso** si es relevante
- **Nombre de estudiante/grupo** si es sobre alguien específico
- **Período de tiempo** si comparas a lo largo del tiempo
- **Lección o módulo** si es sobre contenido específico

## Ejemplos de Conversaciones

### Preparando Reunión con Padres

```
Tú: "Dame un resumen del rendimiento de María García este semestre"

IA: "María García ha completado el 85% de sus lecciones asignadas en 
    Matemáticas 7. Su puntuación promedio es del 78%, con mejor 
    rendimiento en Geometría (92%) y margen de mejora en Álgebra (65%). 
    Típicamente completa las lecciones dentro del tiempo esperado.
    Recomendación: Enfocarse en práctica de ecuaciones algebraicas."
```

### Planificación Semanal

```
Tú: "¿Qué lecciones debo activar la próxima semana para 7A?"

IA: "Basado en el progreso actual de 7A:
    - 78% han completado las Lecciones 1-5
    - Rendimiento promedio: 82%
    - Recomendación: Activar Lecciones 6-7
    - Nota: 3 estudiantes no han completado la Lección 5 aún"
```

### Identificar Estudiantes con Dificultades

```
Tú: "¿Qué estudiantes en mi curso de Ciencias necesitan ayuda?"

IA: "Estudiantes que requieren atención en Ciencias 101:
    1. Pedro Martínez - 45% de finalización, puntuaciones bajas
    2. Ana López - Sin actividad en 2 semanas
    3. Carlos Ruiz - Múltiples intentos fallidos en ejercicios
    
    Recomendación: Programar reuniones individuales"
```

## Qué Puede Acceder la IA

La IA tiene acceso a:

| Tipo de Datos | Ejemplos |
|-----------|----------|
| **Datos de Estudiantes** | Progreso, puntuaciones, actividad |
| **Datos de Cursos** | Estructura, lecciones, ejercicios |
| **Métricas** | Tasas de finalización, tiempo dedicado |
| **Grupos** | Membresía, asignaciones |

### Consideraciones de Privacidad

- La IA solo accede a datos que tienes permiso para ver
- El profesor ve solo sus estudiantes asignados
- El admin ve datos de toda la escuela

## Preguntas de Seguimiento

La IA recuerda el contexto durante la conversación:

```
Tú: "Muéstrame el rendimiento de Pedro en Matemáticas"
IA: [Muestra el rendimiento de Pedro en Matemáticas]

Tú: "¿Qué tal sus calificaciones en Ciencias?"

Tú: "Compáralo con el promedio de la clase"
IA: [Compara a Pedro con la clase - recuerda el contexto]
```

## Generar Reportes

### Reportes Rápidos

Solicita reportes específicos:

```
"Genera un reporte de progreso semanal para 7A"
```

```
"Crea un resumen de finalización de lecciones para Ciencias 101"
```

```
"Muestra el rendimiento de estudiantes por tipo de ejercicio"

Los reportes generados por IA incluyen:
- Estadísticas resumidas
- Información clave
- Recomendaciones

### Usar para Reuniones

**Para reuniones con padres:**
```
"Prepara un resumen amigable para padres para Juan Pérez"
```

**Para administración:**
```
"Genera un reporte de efectividad del curso para este trimestre"
```

## Limitaciones

El Asistente de IA:
- ❌ No puede modificar datos (acceso de solo lectura)
- ❌ No puede activar lecciones o cursos
- ❌ No puede crear o eliminar contenido
- ❌ No tiene datos en tiempo real (ligero retraso)

Para acciones, usa el panel directamente.

## Mejores Prácticas

### Revisión Regular

Usa la IA semanalmente para:
1. Verificar el progreso general
2. Identificar estudiantes con dificultades
3. Planificar activaciones de lecciones
4. Revisar la efectividad del curso

### Antes de Reuniones Importantes

Prepárate con:
1. Resúmenes específicos de estudiantes
2. Comparaciones de progreso
3. Recomendaciones para mejora

### Para Mejora del Curso

Pregunta regularmente:
1. Qué lecciones necesitan trabajo
2. Qué ejercicios son demasiado difíciles/fáciles
3. Cómo mejorar el compromiso

---

## Solución de Problemas

### La IA No Entiende

Intenta:
- Ser más específico
- Incluir nombres y títulos de cursos
- Dividir preguntas complejas en más simples

### No Hay Datos Disponibles

Verifica:
- Los estudiantes han iniciado el curso
- Ha pasado suficiente tiempo para datos
- Tienes acceso a los datos solicitados

### Información Incorrecta

- Verifica los nombres de estudiante/curso
- Verifica el período de tiempo especificado
- Reporta problemas al soporte si persisten
