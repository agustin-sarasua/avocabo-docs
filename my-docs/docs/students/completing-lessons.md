# Completing Lessons

Lessons are the core of your learning experience in PragmaTeach. Each lesson contains exercises that help you learn and practice. This guide shows you how to complete them effectively.

## Starting a Lesson

### Opening a Lesson

1. Navigate to your course
2. Find an available lesson (‚ö™ or üü°)
3. Tap the lesson to open it
4. The lesson screen appears

When you start a lesson, you'll see:

## Exercise Types

### üìù Multiple Choice

**What to do:**
1. Read the question carefully
2. Review all options
3. Tap your answer
4. Submit

**Tips:**
- Read all options before answering
- Look for keywords in the question
- Eliminate obviously wrong answers

### üìñ Reading Comprehension

**What to do:**
1. Read the passage thoroughly
2. Review the questions
3. Refer back to the text as needed
4. Select answers

**Tips:**
- Take your time reading
- Highlight key information mentally
- Answer questions in order

### üéß Listening Comprehension

**What to do:**
1. Tap play to start audio
2. Listen carefully (you can replay)
3. Answer the questions
4. Submit

**Tips:**
- Use headphones for clarity
- Listen at least twice
- Note key details

### üìê Math Problems

**What to do:**
1. Read the problem statement
2. Work through the solution
3. Enter your answer
4. Submit

**Tips:**
- Show your work on paper if needed
- Double-check calculations
- Watch for units

### üé¨ Video Exercises

**What to do:**
1. Watch the video completely
2. Answer any questions
3. Mark as complete

**Tips:**
- Watch in a quiet environment
- Pay attention to key points
- Rewatch if needed

### üìÑ Reading Content

**What to do:**
1. Read through all pages
2. Navigate with Next/Previous
3. Mark as complete when done

**Tips:**
- Take notes if helpful
- Don't rush through
- Review key concepts

## Answering Exercises

### Submitting Answers

1. Select or enter your answer
2. Tap **Submit** or **Check**
3. See if you're correct
4. View feedback

### Correct Answers ‚úÖ

When you answer correctly:
- Green success indicator
- Positive feedback message
- Option to continue

Tap **Continue** to move to the next exercise.

### Incorrect Answers ‚ùå

When you answer incorrectly:
- Feedback explaining the error
- Correct answer shown
- Explanation (if available)

**Learn from mistakes:**
- Read the explanation carefully
- Understand why it's correct
- Remember for next time

### Partial Credit üü°

Some exercises award partial credit:
- Some parts correct
- Feedback on what was right/wrong
- Tips for improvement

## Lesson Completion

### Completing a Lesson

After finishing all exercises:
1. Summary screen appears
2. Shows your results
3. Overall score displayed
4. Option to continue or retry

### Summary Screen

You'll see:
- **Total Exercises** - How many in the lesson
- **Correct Answers** - Number you got right
- **Incorrect Answers** - Number wrong
- **Success Rate** - Your percentage

### Retry Option

If you scored low:
- **Retry Lesson** button (if attempts remaining)
- Start fresh with same exercises
- Try to improve your score

### Attempt Limits

Some lessons limit retries:
- Shows "Attempts: 2 of 3"
- Once exhausted, lesson is locked
- Your best score is recorded

## Troubleshooting

### App Froze During Lesson

- Don't worry, progress is saved
- Close and reopen the app
- Resume the lesson

### Audio/Video Won't Play

- Check your internet connection
- Check device volume
- Try headphones
- Restart the app

### Timer Ran Out

- Lesson is auto-completed
- Your current progress counts
- Retry if attempts remain

### Can't Submit Answer

- Make sure you've selected something
- Check for required fields
- Try refreshing the screen
