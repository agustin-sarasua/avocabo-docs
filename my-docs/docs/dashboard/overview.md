# Dashboard Overview

The PragmaTeach Teacher Dashboard is your central hub for managing educational content, students, and gaining insights into learning progress. This guide will help you navigate and make the most of all features.

## Accessing the Dashboard

1. Go to [app.PragmaTeach.com](https://app.PragmaTeach.com)
2. Sign in with your credentials
3. You'll be taken to your personalized dashboard

## Main Dashboard Screen

When you first log in, you'll see your personalized dashboard with:

### For Teachers

- **Welcome Message** - Personalized greeting with your name
- **Your Metrics** - Quick overview of your teaching activity
  - Active courses count
  - Lessons activated this week
  - Active students
- **Upcoming Live Classes** - Your scheduled video meetings
- **My Courses** - Quick access to your courses
- **Activate Lessons** - Quick widget to activate lessons for students

### For Administrators

- **School Metrics** - Overview of school-wide activity
  - Total users
  - Active courses
  - Student engagement
- **Upcoming Live Classes** - All scheduled video meetings
- **Usage Statistics** - Credit usage and subscription status

## Navigation

The dashboard includes a sidebar navigation with the following sections:

| Section | Description | Who Can Access |
|---------|-------------|----------------|
| **Dashboard** | Main overview screen | All users |
| **Courses** | Manage and create courses | Teachers, Admins |
| **Users** | Manage students, teachers, groups | Admins |
| **Documents** | Upload and manage reference materials | Teachers, Admins |
| **School Settings** | Configure school branding and settings | Admins |
| **Metrics** | Detailed analytics and reports | Teachers, Admins |
| **Chat** | AI Teaching Assistant | All users |
