# Metrics & Analytics

PragmaTeach provides comprehensive analytics to help you understand student progress, identify areas for improvement, and make data-driven teaching decisions.

## Dashboard Metrics

### Teacher Metrics Card

On your dashboard, you'll see:

| Metric | Description |
|--------|-------------|
| **Active Courses** | Number of courses you're teaching |
| **Lessons Activated** | Lessons activated this week |
| **Active Students** | Students who accessed content recently |
| **Completion Rate** | Overall lesson completion percentage |

Click **View More** to access detailed metrics.

### Admin Metrics Card

Administrators see school-wide metrics:

| Metric | Description |
|--------|-------------|
| **Total Users** | All users in the system |
| **Active Courses** | Courses across the school |
| **Student Engagement** | Overall activity level |
| **Usage Statistics** | Credit and resource usage |

## Course-Level Metrics

### Accessing Course Metrics

1. Go to **Courses**
2. Open a course
3. View **Analytics** tab

### Available Metrics

| Metric | What It Shows |
|--------|---------------|
| **Enrollment** | Students assigned to this course |
| **Completion Rate** | % of lessons completed |
| **Average Score** | Mean exercise performance |
| **Time Spent** | Average time per lesson |
| **Drop-off Points** | Where students stop |

### Interpreting Course Data

**High completion, high scores:** ✅ Course is effective

**High completion, low scores:** ⚠️ Content may be too easy or not challenging enough

**Low completion, high scores:** ⚠️ Engaged students do well, but many don't start

**Low completion, low scores:** ❌ Course needs review

## Using Metrics Effectively

### Regular Review Schedule

| Frequency | What to Review |
|-----------|----------------|
| **Daily** | New completions, immediate issues |
| **Weekly** | Progress trends, activation planning |
| **Monthly** | Course effectiveness, student progress |
| **Quarterly** | Strategic improvements, curriculum changes |

### Action Items from Metrics

**Low completion rate:**
1. Check if content is too difficult
2. Verify lessons are activated
3. Review student engagement

**Low scores on specific exercises:**
1. Review exercise clarity
2. Check if content was covered
3. Consider adding hints/explanations

**Student inactivity:**
1. Reach out personally
2. Check for technical issues
3. Assess if course is appropriate

## Understanding Score Calculations

### How Scores Work

| Component | Weight |
|-----------|--------|
| **Exercise Correctness** | Primary factor |
| **Attempts Used** | Some exercises track attempts |
| **Completion** | Binary (done/not done) |

### Score Interpretation

| Score | Interpretation |
|-------|----------------|
| **90-100%** | Excellent understanding |
| **70-89%** | Good, some review needed |
| **50-69%** | Needs additional practice |
| **Below 50%** | Significant support needed |
