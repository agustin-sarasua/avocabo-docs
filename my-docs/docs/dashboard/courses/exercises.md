## AI-Generated Exercises

### How It Works

When you create a course, the AI:
1. Analyzes lesson content
2. Determines appropriate exercise types
3. Generates questions and answers
4. Creates variety across lessons

### Exercise Distribution

The AI balances exercise types:
- Mix of question-based and content-based exercises
- Appropriate difficulty for target audience
- Variety to maintain engagement

## Creating Exercises Manually

### From the Dashboard

1. Click the **+** floating action button
2. Select **Create Exercise**
3. Choose the exercise type
4. Fill in the exercise details
5. Assign to a lesson

### From Lesson Details

1. Open a lesson
2. Go to the **Exercises** tab
3. Click **Add Exercise**
4. Complete the exercise form

### Exercise Form Fields

| Field | Description |
|-------|-------------|
| **Type** | Select exercise type |
| **Title** | Name of the exercise |
| **Instructions** | What students should do |
| **Content** | Questions, text, media |
| **Correct Answer(s)** | The right answer(s) |
| **Explanation** | Why the answer is correct |
| **Language** | Exercise language |

## Editing Exercises

### Accessing Exercise Details

1. Navigate to the lesson
2. Find the exercise
3. Click to open details

### What You Can Edit

- Title and instructions
- Question content
- Answer options
- Correct answer designation
- Explanations
- Associated media

### Saving Changes

Click **Save** after making edits. Changes are reflected immediately for students who haven't started the exercise.

:::note
Students who have already attempted an exercise will see their original version.
:::

## Viewing Student Responses

### Response Summary

For each exercise, view:
- Total attempts
- Correct rate
- Average time spent
- Individual responses

### Detailed Responses

Click **View Responses** to see:
- Each student's answer
- Whether it was correct
- Time taken
- Attempt timestamp

### Identifying Struggling Students

Look for:
- Multiple incorrect attempts
- Very long completion times
- No attempts (student may be stuck)

## Exercise Analytics

### Performance Metrics

View for each exercise:

| Metric | Description |
|--------|-------------|
| **Completion Rate** | % of students who completed |
| **Correct Rate** | % of correct answers |
| **Avg Time** | Average time to complete |
| **Attempts** | Total number of attempts |

### Identifying Problem Exercises

Low correct rates may indicate:
- Question is too difficult
- Question is confusingly worded
- Content wasn't covered adequately
- Technical issue with exercise

### Using Insights

1. Review exercises with low success rates
2. Check if the content needs clarification
3. Consider adding hints or explanations
4. Ask the AI Assistant for recommendations

## Troubleshooting

### Exercise Not Displaying

Check:
- Lesson is activated
- Media files are uploaded correctly
- No HTML/formatting errors

### Students Report Wrong Answers

1. Review the exercise
2. Verify correct answer is marked
3. Check for any ambiguity
4. Update if needed and communicate with students

### Audio/Video Not Playing

- Check file format compatibility
- Verify URL is accessible
- Test on different devices
- Re-upload if necessary
