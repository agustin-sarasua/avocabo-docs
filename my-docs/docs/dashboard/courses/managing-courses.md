# Managing Courses

Once you've created courses, you'll need to manage them effectively. This guide covers all aspects of course management.

## Viewing Your Courses

### Course List

Navigate to **Courses** in the sidebar to see all your courses.

**What you'll see:**
- Course title and description
- Course image
- Active/Inactive status
- Favorite star for quick access

### Filtering and Searching

Use the search bar to find courses by:
- Course title
- Description keywords

### Course Favorites

Mark frequently accessed courses as favorites:
1. Click the ⭐ star icon on any course
2. Favorited courses appear at the top of your list

---

## Course Details

Click on any course to view its details:

### Overview Tab

- **Course Information** - Title, description, subject
- **Status** - Active or Inactive
- **Groups** - Assigned student groups
- **Teachers** - Assigned teachers
- **Statistics** - Student enrollment and progress

### Module Structure

View and manage the course structure:
- Expand modules to see lessons
- Drag and drop to reorder modules
- View lesson count per module

---

## Editing Courses

### Edit Basic Information

1. Open course details
2. Click **Edit** button
3. Modify:
   - Course title
   - Description
   - Subject
   - Target audience

### Reorganize Modules

Drag and drop modules to change their order:
1. Click and hold the module drag handle
2. Drag to new position
3. Release to confirm

### Manage Assigned Groups

Control which student groups can access the course:

1. Go to **Groups** section in course details
2. Click **Manage Groups**
3. Add or remove groups as needed

:::note
Students only see courses assigned to their groups.
:::

### Manage Teachers

Add or remove teachers from a course:

1. Go to **Teachers** section
2. Click **Add Teachers**
3. Select teachers from the list
4. Added teachers can now manage this course

---

## Course Actions

### Activate/Deactivate Course

Toggle course visibility:

| Status | Students Can See | Can Edit Content |
|--------|------------------|------------------|
| **Active** | ✅ Yes | ✅ Yes |
| **Inactive** | ❌ No | ✅ Yes |

To toggle:
1. Open course details
2. Click the **Activate** or **Deactivate** button

### Duplicate Course

Create a copy of an existing course:

1. Open course list
2. Click the menu (⋮) on the course
3. Select **Duplicate**
4. Edit the copy as needed

### Delete Course

:::danger Caution
Deleting a course is permanent and removes all associated data including student progress.
:::

1. Open course list
2. Click the menu (⋮) on the course
3. Select **Delete**
4. Confirm deletion in the dialog

## Working with Modules

### View Module Details

Click on any module to see:
- Module title and description
- List of lessons
- Module image
- Topics covered

### Edit Module

1. Click the **Edit** button on a module
2. Modify:
   - Title
   - Description
   - Topics
   - Image

### Reorder Lessons Within Module

1. Open the module
2. Drag lessons to reorder
3. New order is saved automatically

---

## Working with Lessons

### View Lesson Details

Click on any lesson to see:
- Title and description
- Lesson image
- Associated exercises
- Lesson metrics
- Document references

### Edit Lesson

1. Click **Edit** on a lesson
2. Modify:
   - Title
   - Description
   - Topics
   - Image

### View Lesson Exercises

In the lesson details, see all exercises:
- Exercise type and description
- Student responses
- Performance statistics

### Lesson Metrics

View detailed analytics:
- Completion rate
- Average time spent
- Exercise performance
- Student attempts

## Course Reports

### Generate Course Report

Get a summary of course performance:

1. Open course details
2. Click **Reports**
3. Select report type:
   - **Progress Report** - Student completion status
   - **Performance Report** - Exercise scores
   - **Time Report** - Time spent by students

### Using the AI Assistant

Ask the AI Teaching Assistant for insights:

**Example questions:**
- "How effective is my Math 7 course?"
- "Which lessons are most difficult for students?"
- "What can I improve in this course?"

## Best Practices

### Course Organization

| Do | Don't |
|----|-------|
| Use clear, descriptive module names | Use "Module 1, Module 2" |
| Keep lessons focused on single topics | Combine unrelated topics |
| Order content logically | Random topic ordering |

### Course Maintenance

| Do | Don't |
|----|-------|
| Review course content regularly | Set and forget |
| Update based on student feedback | Ignore performance data |
| Keep exercises varied | Use only one exercise type |

### Group Management

| Do | Don't |
|----|-------|
| Assign appropriate groups | Make all courses public |
| Review group access periodically | Forget to update access |
| Create group-specific pacing | One pace for everyone |
